# 🧠 Explorando o Universo da Inteligência Artificial

Bem-vindo ao meu laboratório de IA! Este repositório é um diário da minha jornada de aprendizado no  mundo das redes neurais. Aqui, eu exploro, testo e documento meus estudos sobre a construção e avaliação de modelos de Inteligência Artificial.

## 🎯 Objetivo

O principal objetivo deste projeto é aprofundar o conhecimento prático em três pilares fundamentais do Deep Learning

1.  **Arquitetura de Redes Neurais:** Entender como projetar e estruturar modelos, escolhendo o número de camadas, neurônios e funções de ativação.
2.  **Otimização de Hiperparâmetros:** Aprender a ajustar os "botões" do treinamento, como a taxa de aprendizado, o número de épocas e o otimizador, para alcançar os melhores resultados.
3.  **Métricas de Avaliação:** Ir além da acurácia, compreendendo e analisando criticamente métricas como **Precision**, **Recall**, **F1-Score** e a **Curva ROC** para validar a real eficácia de um modelo.

## 🛠️ Tecnologias Utilizadas

Este projeto é desenvolvido em um ambiente **Jupyter Notebook**, que permite uma exploração interativa de código, visualizações e texto. As principais bibliotecas utilizadas são:

*   **PyTorch:** Um poderoso framework de Deep Learning para construir e treinar as redes neurais.
*   **Scikit-learn:** Essencial para o pré-processamento dos dados, divisão do dataset e cálculo das métricas de avaliação.
*   **Pandas:** Para a manipulação e análise inicial dos dados.
*   **Matplotlib & Seaborn:** Para a criação de visualizações claras e informativas, como as curvas de aprendizado e matrizes de confusão.
